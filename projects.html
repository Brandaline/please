My Projects

While this is just a glimpse into my work, I’ve created many other projects, each showcasing my passion for learning, adapting, and solving unique challenges through technology.

Card Matching Game

This card-matching game blends creative design and functional coding to provide an engaging interactive experience. Featuring cards representing different coding languages, it challenges players to match pairs while keeping track of their performance. The game records the number of flips and matches, with a convenient reset button at the bottom for quick restarts. Designed in Photoshop and built using HTML, CSS, and JavaScript, this project highlights my ability to combine visual design with front-end development. Its responsive and intuitive design reflects my focus on creating seamless, user-friendly web applications.

Mad Lib Generator

This Mad Libs generator is a fun and interactive project built using HTML, CSS, and JavaScript. It allows users to input their own creative words—nouns, verbs, adjectives, and more—and generates a unique, often humorous story based on their entries. The generator features a user-friendly interface and the option to quickly create new stories, ensuring endless entertainment. This project highlights my ability to design and develop engaging web applications that blend logic, interactivity, and creativity seamlessly.

Airbnb Search Engine

As part of my global career accelerator, I worked on enhancing an Airbnb search engine by optimizing JavaScript functions to improve the user experience. I added functionality to specify the type of place (e.g., house, apartment, shared room), as well as set minimum and maximum price ranges for a more tailored search. This project showcases my ability to modify and extend existing codebases to meet user needs while demonstrating strong problem-solving and JavaScript programming skills. The result is a more dynamic and customizable search experience that reflects real-world application development.
